Cómo comprimir imágenes para web

Una imagen grande puede hacer que una página tarde más en cargar. La compresión reduce el peso del archivo manteniendo una apariencia razonable para el tamaño en que se verá.

Empieza por las dimensiones

Si una foto se muestra a 900 px de ancho, no suele tener sentido subirla a 4000 px. Primero redimensiona, después comprime.

Ajusta la calidad

Para fotografías, una calidad entre 70 y 85 suele equilibrar bien peso y nitidez. En imágenes con texto pequeño conviene revisar el resultado con más cuidado.

Usa WebP cuando encaje

WebP puede reducir bastante el peso frente a JPG manteniendo una calidad visual similar en navegadores modernos.
