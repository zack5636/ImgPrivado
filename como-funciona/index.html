Client-side

Cómo funciona imgprivado

Las herramientas usan APIs del navegador para leer, dibujar y descargar imágenes localmente.

1. Eliges un archivo

El navegador recibe el archivo desde tu dispositivo. imgprivado no lo sube a ningún servidor.

2. Canvas procesa la imagen

Según la herramienta, Canvas comprime, cambia tamaño, recorta, rota o convierte el formato.

3. Descargas el resultado

Se crea un Blob local y se dispara una descarga desde tu navegador.
